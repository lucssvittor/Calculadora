A **Calculadora** é uma calculadora minimalista com um design moderno e interface intuitiva. Desenvolvida para proporcionar uma experiência fluida e visualmente agradável para os usuários.

## 🔧 Tecnologias Utilizadas

- **HTML, CSS e JavaScript** para a interface e lógica  
- **Design responsivo** para compatibilidade com diferentes dispositivos  

## 📌 Funcionalidades

✔️ Interface estilizada com tema escuro  
✔️ Operações matemáticas básicas (+, -, *, /, %)  
✔️ Animações suaves na interação
